# CSM Buea

Next.js 15 foundation for CSM Buea — quality electronics, household appliances, and home decor essentials delivered across Cameroon.

Built with Next.js, TypeScript, Tailwind CSS, Supabase, and Prisma.

## Requirements

- Node.js 22.11+

## Getting started

1. Install dependencies: `npm install`
2. Copy `.env.local.sample` to `.env.local` and fill in the values
3. Run the dev server: `npm run dev`
4. Open http://localhost:3000
